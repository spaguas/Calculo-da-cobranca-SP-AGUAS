# Cálculo da cobrança SP-ÁGUAS
Este projeto visa ao desenvolvimento de uma calculadora para a cobrança pelo uso da água no Estado de São Paulo. A cobrança pelo uso da água segue um decreto específico vigente para cada bacia hidrográfica. Os valores a serem cobrados, as ponderações, os coeficientes, o procedimento de cobrança, as fórmulas, entre outros aspectos, podem variar de acordo com o decreto vigente em cada Unidade de Gerenciamento de Recursos Hídricos (UGRHI). Nesse sentido, este projeto busca facilitar a visualização, pelo usuário, dos valores cobrados, sem que ele precise realizar a leitura minuciosa de cada decreto de seu interesse.
